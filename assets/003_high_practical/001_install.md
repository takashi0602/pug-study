# インストール

[Node.js](https://nodejs.org/ja/download/)をインストールする.  
Node.jsとは, サーバサイドで動くJavaScript.

> 準備
```bash
$ mkdir pug-test
$ cd pug-test
```

> npm インストール  
パッケージ管理システム
```bash
$ npm init
$ npm install (npm i)
```

> gulp インストール  
さまざまな作業を自動化するビルドシステム
```bash
$ npm i gulp --save-dev
```

> Pug インストール  
HTMLが簡単に書ける 
```bash
$ npm i gulp-pug --save-dev
```
