# 変数
Pugでは変数を定義することができる.

```
- var title = "ゴリラ";
h1 #{title}
```

```html
<h1>ゴリラ</h1>
```

変数が使えることの利点は, 例えば以下のように同じ値が何度も記述される場合に, HTMLでは何度も同じ値を記述しなければならない.

```html
<ul>
  <li>たけし</li>
  <li>たけし</li>
  <li>たけし</li>
  <li>たけし</li>
</ul>
```

この場合に`たけし`を`たかし`に変更することになったとする.  
すると, HTMLの場合は５つ全て変更する必要があるが, Pugでは以下のように

```
- val name = "たけし";
ul
  li #{name}
  li #{name}
  li #{name}
  li #{name}
```

とできるため, 変数nameの値を１箇所変更するだけで済む.
