# 改行
`br`でテキストを改行したあとに続けてテキストを表示させたい場合には`|`を使う.

> Pug
```
p
  メロスは激怒した。
  br
  | 必ず、かの邪智暴虐の王を除かなければならぬと決意した。
  br
  | メロスには政治がわからぬ。
```

これをHTMLに変換すると以下のようになる.

> HTML
```html
<p>
  メロスは激怒した。<br>
  必ず、かの邪智暴虐の王を除かなければならぬと決意した。<br>
  メロスには政治がわからぬ。
</p>
```

ちなみに, 以下の例は上手く動作しない.

> Pug
```
p
  メロスは激怒した。
  br 必ず、かの邪智暴虐の王を除かなければならぬと決意した。
  br メロスには政治がわからぬ。
```
